# 在海康做什么？

## vhatom

### vhatom是一个什么平台？

-   是一个开发移动应用 iaas平台。并且包括了硬件设备应用

### 有什么用处

-   简化开发app流程。将app功能模块拆分。
-   服务提供开发者可以上传模板，壳工程，组件供移动应用app开发者选取，各自的逻辑则以H5资源包的形式上传。通过后台的编译打
    包，集成发布在hikyun上。提供安装包
-   该平台把可以把H5包集成到app上
